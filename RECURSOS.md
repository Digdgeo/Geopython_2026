# Recursos para seguir aprendiendo Python y GeoPython

Una recopilación de recursos para continuar tu camino con Python geoespacial, desde los fundamentos hasta análisis avanzado con datos raster, vectoriales y satélite.

---

## Python general

- [Anaconda - Get Started](https://freelearning.anaconda.cloud/get-started-with-anaconda) — Introducción a Python con el ecosistema Anaconda
- [Python Data Science Handbook](https://jakevdp.github.io/PythonDataScienceHandbook/) — Libro completo y gratuito de Jake VanderPlas (NumPy, Pandas, Matplotlib, scikit-learn)
- [Real Python](https://realpython.com/) — Tutoriales y artículos de calidad sobre Python práctico

---

## Cursos y programas completos

- [Geo-Python (University of Helsinki)](https://geo-python-site.readthedocs.io/en/latest/) — Curso universitario gratuito de Python orientado a ciencias de la tierra
- [AutoGIS (University of Helsinki)](https://autogis-site.readthedocs.io/en/latest/index.html) — Continuación del anterior, análisis GIS con Python
- [Earth Data Science (CU Boulder)](https://www.earthdatascience.org/workshops/gis-open-source-python/) — Talleres de GIS open source con Python
- [EO College](https://eo-college.org/courses/) — Cursos de observación de la Tierra (teledetección, SAR, etc.)
- [Project Pythia](https://projectpythia.org/) — Recursos educativos para Python en ciencias de la atmósfera y geociencias
- [Pangeo Tutorial Gallery](https://gallery.pangeo.io/) — Notebooks de análisis geoespacial a escala con Xarray y Dask

---

## Librerías vectoriales

- [GeoPandas - User Guide](https://geopandas.org/en/stable/docs/user_guide/io.html) — Lectura/escritura de datos vectoriales con GeoPandas
- [Shapely Manual](https://shapely.readthedocs.io/en/stable/manual.html) — Geometrías y operaciones espaciales en Python
- [PyProj](https://pyproj4.github.io/pyproj/stable/) — Transformaciones de sistemas de referencia de coordenadas
- [Folium](https://python-visualization.github.io/folium/) — Mapas web interactivos con Leaflet desde Python

---

## Librerías raster

- [Rasterio](https://rasterio.readthedocs.io/en/latest/) — Lectura y escritura de datos raster georreferenciados
- [Rioxarray](https://corteva.github.io/rioxarray/stable/) — Extensión geoespacial para Xarray (raster multidimensional)
- [Xarray](https://docs.xarray.dev/en/stable/) — Arrays N-dimensionales con etiquetas, ideal para datos de satélite
- [EarthPy](https://earthpy.readthedocs.io/en/latest/) — Utilidades para datos raster y vectoriales orientadas a Earth Science

---

## Google Earth Engine y teledetección

- [Geemap](https://geemap.org/) — Python + GEE: mapas interactivos, análisis de series temporales, exportación
- [YouTube @giswqs (Qiusheng Wu)](https://www.youtube.com/@giswqs) — Canal del autor de Geemap y Leafmap, tutoriales muy prácticos
- [Leafmap](https://leafmap.org/) — Mapas interactivos sin GEE, integra múltiples backends (ipyleaflet, folium, plotly)
- [Google Earth Engine Python API](https://developers.google.com/earth-engine/guides/python_install) — Documentación oficial del API de Python para GEE

---

## Datos y plataformas

- [Copernicus Data Space](https://dataspace.copernicus.eu/) — Acceso gratuito a imágenes Sentinel (ESA)
- [Microsoft Planetary Computer](https://planetarycomputer.microsoft.com/) — Catálogo STAC con datos satelitales y entorno de análisis en la nube
- [OpenEO Platform](https://openeo.cloud/) — API estándar para procesar datos de observación de la Tierra en la nube
- [NASA Earthdata](https://earthdata.nasa.gov/) — Portal de acceso a datos satelitales de la NASA (Landsat, MODIS, etc.)
- [NDVI2GIF](https://github.com/Digdgeo/ndvi2gif) — Generación de GIFs animados de NDVI a partir de imágenes Sentinel-2

---

## Comunidad y foros

- [GIS Stack Exchange](https://gis.stackexchange.com/) — El mejor sitio de preguntas y respuestas sobre GIS y Python geoespacial
- [r/gis](https://www.reddit.com/r/gis/) y [r/learnpython](https://www.reddit.com/r/learnpython/) — Comunidades activas en Reddit
- [Pangeo Discourse](https://discourse.pangeo.io/) — Foro de la comunidad Pangeo para ciencia geoespacial en Python

---

> **Consejo:** La mejor forma de aprender es con datos reales. Descarga una imagen de tu zona de interés en el [Copernicus Data Space](https://dataspace.copernicus.eu/) o usa el [Planetary Computer](https://planetarycomputer.microsoft.com/) y pon en práctica estas herramientas.
